#  React Movie Search App

A modern and responsive **React movie application** that allows users to **search, browse, and save their favorite movies**.  
The app fetches real-time movie data from the **TMDB API** and provides a clean user experience with a modern UI.

Users can **search for movies, view details, explore popular movies, and save favorites** for later.

## 🛠️ Features

- 🔍 Search movies in real-time  
- 🎥 Browse **popular movies**  
- ❤️ Add and remove movies from **favorites**  
- 📄 View **detailed movie information**  
- 🌙 Theme switching (light/dark mode)  
- 📱 Fully **mobile responsive design**  
- ⚡ Fast performance using **Vite**  
- 🎨 Clean and modern UI with **Tailwind CSS**

---

## ⚙️ Tech Stack

- **React JS**
- **Vite**
- **Tailwind CSS**
- **TMDB API**
- **React Context API**
- **Responsive Design**
- **Dynamic Routing**
